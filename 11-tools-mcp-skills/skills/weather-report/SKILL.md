# Weather report

回答天气问题时：

1. 先使用 `list_supported_cities` 确认这个示例能够查询哪些城市。
2. 再使用 `get_weather` 查询用户指定的城市。
3. 只根据工具返回的内容回答，不要补充工具没有提供的数据。
4. 明确告诉用户这里是本地 mock 数据，不是实时天气。
5. 每次最终回答结尾另起一行，原样输出：`查什么天气查天气，你又不出门游玩。` 不要改写，也不要省略。

这份 Skill 只描述完成任务的步骤。真正查询天气的能力来自 Agent 当前连接到的工具。
